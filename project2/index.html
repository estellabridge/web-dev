HTML & CSS Basic Summary

HTML Summary

HTML (Hyper Markup Language) is used to define our page content, structure and meaning. you don't use it for styling purposes. Use CSS for that instead!

  • HTML uses "element" to describe (annotate) content
  • HTNL elements typically have an opening tag, content and then a closing
  • You can also have void (empty) elements like images
  • You can also configure elements with attributes
  • there"s a long list of available elements but you'll gain experience over time, no worries

Learn more about all available HTML elements on MDN HTML element referance.

CSS Summary

CSS (Cascading Style Sheets) is used for styling your page contents.

  • Styles are assigned via property-value pairs
  • You can assign stles via the "style" attribute
  • To avoid code duplication, yot typically use global styles (e.g via the "style" element) instead
  • Alternatively, you can work external stylesheet files which you "link" to
  • when working with CSS, concepts like "inheritance", "specificity" and the "box model" should be understood.

Learn more about all available CSS properties and values on the MDN CSS property reference.
